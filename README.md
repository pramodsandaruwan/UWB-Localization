# UWB-Localization
REAL-TIME 3D OUTDOOR LOCALIZATION USING UWB, GNSS AND IMU FUSION WITH DEEP LEARNING
## Goal of Research:
The objective of the thesis is to design, implement and evaluate a real-time three-dimensional outdoor localization system that combines Ultra-Wideband, Global Navigation Satellite System and Inertial Measurement Unit data using a deep learning-based fusion architecture, with high positioning accuracy compared to classical filter-based methods in a dynamic outdoor setting.
## Objectives:
Develop a real-time 3D outdoor localization system by fusing GNSS, UWB, and IMU data
Mitigate single-sensor limitations, including GNSS signal degradation, UWB range constraints, and IMU drift
Implement a Temporal Convolutional Network (TCN) to enhance UKF-based sensor fusion and improve position estimation accuracy.
Evaluate system performance in challenging outdoor environments, including GPS-denied and multipath conditions
